# Git Commands

This repository contains explanations and examples of common Git commands used for version control and collaboration in software development.

The purpose of this repo is to help beginners learn how to use Git by providing descriptions, syntax, and examples of important commands and flags.

## Files

* `README.md` — Overview of the repository
* `commands.md` — Detailed explanations of Git commands and flags

## Topics Covered

* Cloning repositories
* Creating and switching branches
* Checking repository status
* Adding and committing changes
* Pulling and pushing code
* Merging branches
* Useful Git flags

Git is an essential tool for developers, and this repository serves as a quick reference guide for commonly used commands.
